# Sistema de Gestión de Equipos de Protección Personal (EPP)

Proyecto desarrollado para el curso de Desarrollo Web.

## Tecnologías

### Frontend
- Angular
- TypeScript
- Angular Router
- Reactive Forms
- Pipes
- HttpClient

### Backend
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- MySQL

## Funcionalidades

- Inicio de sesión con JWT
- Dashboard Industrial
- Gestión de Cascos
- Gestión de Guantes
- Gestión de Lentes
- Gestión de Mascarillas
- Gestión de Zapatos
- Consumo de API REST
