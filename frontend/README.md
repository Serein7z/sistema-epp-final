Frontend

Este proyecto fue generado utilizando Angular CLI versión 20.3.31.

Servidor de desarrollo

Para iniciar un servidor de desarrollo local, ejecuta:

ng serve

Una vez que el servidor esté en ejecución, abre tu navegador y accede a:

http://localhost:4200/

La aplicación se recargará automáticamente cada vez que modifiques alguno de los archivos fuente.

Generación de código (Code Scaffolding)

Angular CLI incluye potentes herramientas para generar código automáticamente. Para crear un nuevo componente, ejecuta:

ng generate component nombre-del-componente

Para ver la lista completa de los esquemas disponibles (como componentes, directivas o pipes), ejecuta:

ng generate --help
Compilación del proyecto

Para compilar el proyecto, ejecuta:

ng build

Esto compilará la aplicación y almacenará los archivos generados en el directorio:

dist/

De forma predeterminada, la compilación para producción optimiza la aplicación para ofrecer un mejor rendimiento y mayor velocidad.

Ejecución de pruebas unitarias

Para ejecutar las pruebas unitarias utilizando el ejecutor de pruebas Karma, usa el siguiente comando:

ng test
Ejecución de pruebas de extremo a extremo (End-to-End)

Para ejecutar las pruebas de extremo a extremo (e2e), utiliza:

ng e2e

Angular CLI no incluye un framework de pruebas e2e de forma predeterminada, por lo que puedes elegir el que mejor se adapte a las necesidades de tu proyecto.
